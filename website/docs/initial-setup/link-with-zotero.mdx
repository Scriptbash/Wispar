# Link With Zotero

Wispar has a very simple Zotero integration. If you connect your Zotero account to Wispar, you will be able to send articles directly to your Zotero account. The articles are sent into a collection called "Wispar". If the collection does not exist in your account, one will be created the first time you send an article. 

## Method 1 – During On-boarding

<img src="/img/docs/app_setup/zotero/link_zotero.png" alt="Screenshot showing the Zotero account linking screen" height="500"/>

1. Tap the <b>Link my Zotero account</b> button

<img src="/img/docs/app_setup/zotero/link_zotero_button.png" alt="Screenshot showing the link my zotero button" width="300"/>

2. Tap the <b>Create a new API key</b> button

<img src="/img/docs/app_setup/zotero/create_zotero_api_key.png" alt="Screenshot showing the button to create an API key" width="300"/>

3. <b>Sign in</b> to your Zotero account. You will then be redirected to the Zotero API key generation page.

4. Start by typing a <b>name</b> for your API key.

<img src="/img/docs/app_setup/zotero/zotero_api_key_name.png" alt="Screenshot showing the API key name field" height="100"/>

5. Next, check both "<b>Allow library access</b>" and "<b>Allow write access</b>". Without write access, Wispar will <b>not</b> be able to send articles into your account.

<img src="/img/docs/app_setup/zotero/zotero_permissions_1.png" alt="Screenshot showing the two required permissions" height="200"/>

6. Leave the group permissions to their default values or select "<b>None</b>".

7. You can leave the specific groups permissions <b>unchecked</b>.

<img src="/img/docs/app_setup/zotero/zotero_overview.png" alt="Screenshot showing the overview of the API key generation" height="500"/>

8. Tap on the <b>Save</b> button. Your API key will be revealed. You are almost done!

<img src="/img/docs/app_setup/zotero/zotero_save_button.png" alt="Screenshot showing the save button" height="50"/>

9. <b>Copy your API key</b> and <b>return</b> to Wispar.
10. <b>Paste your API key</b> into the text field.
11. Tap on the "<b>Save</b>" button.

<img src="/img/docs/app_setup/zotero/zotero_settings_screen.png" alt="Screenshot showing the API key entered in Wispar" height="500"/>

:::tip[Tip]
If you see a successful message, you are all done! If an error message was shown, please <b>verify your API key</b>. Also double check you have set the <b>correct permissions</b> when creating the key.
:::

## Method 2 – From Settings

If you have already completed on-boarding, you can still link your Zotero account through the app settings.

1. From the <b>Home</b> screen:  
   <img src="/img/docs/home_bottom_menu.png" alt="Screenshot showing the home screen as the active screen" width="300"/>

2. Tap the Wispar <img src="/img/wispar.png" alt="Screenshot showing the Wispar icon" width="20"/> icon at the top-right corner and select <b>Settings</b>.  
   <img src="/img/docs/wispar_menu.png" alt="Screenshot showing the Wispar menu open" width="300"/>

3. Tap the <b>Zotero</b> tile and follow the steps XX from Method 1 from step 2.  
   <img src="/img/docs/app_setup/zotero/zotero_tile.png" alt="Screenshot showing the Zotero settings" width="100"/>
